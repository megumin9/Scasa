122/**
123 * @anchor color_tags
124 * @name color tags
125 *
126 * Most print function have only one color parameter (or none at all), therefore they would
127 * print the whole text in one color. To print some parts of a text in a different color,
128 * one would have to write separate calls to the print functions.
129 *
130 * Color tags allow embedding coloring instructions directly into the text, which allows
131 * a single call to the print function with the full text. The color tags are removed
132 * (not printed), but the text inside the tags is printed with a specific color.
133 *
134 * Example: Print "You need a tool to do this." in white and the word "tool" in red.
135 * \code
136 * wprintz( w, c_white, "You need a " );
137 * wprintz( w, c_red, "tool" );
138 * wprintz( w, c_white, " to do this." );
139 * \endcode
140 * Same code with color tags:
141 * \code
142 * print_colored_text( w, c_white, "You need a <color_red>tool</color> to do this." );
143 * \endcode
144 *

148 * not contain any other color tags.
149 *

161 * length of a string containing color tags. The parameter instructs `utf8_width` to ignore the

692 * @param string_for Function that accepts an element and returns a representing string.
